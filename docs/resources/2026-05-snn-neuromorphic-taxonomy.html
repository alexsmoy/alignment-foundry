I
Biological Foundations
1952 – 1990s
1952
Hodgkin-Huxley Model
H-H conductance model
Nobel Prize–winning biophysical model of action potential generation using voltage-gated ion channel conductances (Na⁺, K⁺). The ground truth of all neuron models.
Biophysical4-variable ODE
Key Contribution
Derived from giant squid axon voltage-clamp experiments. Four coupled ODEs model membrane potential V, Na activation m, Na inactivation h, K activation n. Sets the biological fidelity ceiling for all simplified models.
Legacy
Every LIF, Izhikevich, and AdEx model is an approximation of H-H. H-H simulations are still used in detailed cortical column models (Blue Brain Project, Human Brain Project).
Use Cases
  • Neuroscience simulation
  • Drug target modeling
  • Retinal prosthetics
  • Cardiac pacemaker design
1961 / 1990s
Leaky Integrate-and-Fire
LIF / IF / Perfect IF
Simplifies H-H to a single RC circuit equation. Membrane potential leaks toward rest; fires when threshold θ reached, then resets. The workhorse neuron model for SNNs.
Neuron Model1-variableTractable
Variants
Exponential LIF (AdEx), Quadratic IF, Izhikevich (2003) — 2-variable model capturing 20+ firing patterns with low compute. LIF is used in nearly all neuromorphic chip designs.
Use Cases
  • Neuromorphic hardware neurons
  • SNN inference on Loihi
  • Computational neuroscience
  • Real-time edge inference
1972 – 1990
Neural Coding Theories
Rate / Temporal / Population
How do neurons encode information in spikes? Rate coding: mean firing rate. Temporal coding: precise spike timing (first spike, ISI). Population coding: distributed patterns across neurons.
CodingTheoryInformation
Coding Schemes
Rate coding (Adrian, 1926): firing frequency encodes stimulus strength. Time-to-first-spike (TTFS): latency encodes intensity — 10–100× more energy-efficient. Phase coding: relative to oscillation. Burst coding: pattern of bursts carries info. Sparse coding: few active neurons at once (biological cortex ~1–5% active).
Importance for SNNs
Choice of coding scheme fundamentally determines SNN power efficiency. TTFS enables single-timestep inference. Rate codes are easiest to convert from ANNs.
1949
Hebbian Learning
“Neurons that fire together, wire together”
Hebb’s postulate: synaptic weight strengthens when pre- and post-synaptic neurons fire simultaneously. The conceptual ancestor of all local learning rules in SNNs and Hopfield networks.
Learning RuleLocalUnsupervised
Legacy
Directly inspired STDP (see Era II). Underlies Hopfield networks (1982), Boltzmann machines, and modern contrastive Hebbian learning. The BCM rule (Bienenstock-Cooper-Munro, 1982) added a sliding threshold preventing runaway potentiation.
Use Cases
  • Associative memory
  • Unsupervised feature learning
  • On-chip learning
II
First-Generation SNNs & STDP
1996 – 2007
1998 – 2002
Spike-Timing Dependent Plasticity
STDP
If pre-synaptic neuron fires before post-synaptic (causal), synapse potentiates (LTP). If after (anti-causal), synapse depresses (LTD). Asymmetric Hebbian rule based on millisecond timing.
Learning RuleTemporalUnsupervised
Key Contribution
Experimentally confirmed in hippocampal and cortical slices (Bi & Poo 1998, Markram 1997). Implements causal Hebbian learning with a biological substrate. Variants: R-STDP (reward-modulated), triplet STDP, voltage-dependent STDP.
Use Cases
  • Unsupervised feature extraction
  • On-chip online learning (Loihi)
  • Temporal pattern recognition
  • Sequence learning
Limitation
Hard to scale to deep networks; typically trains only 1–2 layers effectively. Supervised equivalents (e-prop, DECOLLE) needed for complex tasks.
1996 – 2002
SpikeProp & Tempotron
Early supervised SNN learning
SpikeProp (Bohte 2002): first backpropagation through spike times using exact spike-timing gradients. Tempotron (Gütig 2006): single neuron classifier that learns to fire / not-fire based on input spike patterns.
SupervisedSpike TimingLearning Rule
Key Contribution
First demonstration that gradient-based learning is possible in spiking networks with temporal coding. Limited to small shallow networks but opened the theoretical path to surrogate gradient methods.
Use Cases
  • Temporal pattern classification
  • Auditory processing
  • Proof-of-concept SNN training
1997 – 2004
Liquid State Machines
LSM (Maass 2002)
Randomly connected recurrent SNN acts as a “liquid” — projects inputs into a high-dimensional dynamic state. A linear readout layer classifies the transient trajectory. Reservoir computing with spikes.
ArchitectureReservoirRecurrent
Key Contribution
Separated memory (liquid / reservoir) from computation (readout). Only readout is trained — the recurrent core is random or fixed. Equivalent to Echo State Networks (ESN) for rate neurons. Edge-of-chaos dynamics gives best performance.
Use Cases
  • Speech recognition (early)
  • Motor control
  • Time-series prediction
  • Neuromorphic sensor processing
2005
Polychronization
Izhikevich 2006
In networks with axonal conduction delays, groups of neurons can fire in precise repeating temporal patterns (polychronous groups). Exponential number of coexisting patterns in a single network — a proposed basis for memory capacity.
TheoryMemoryTemporal Patterns
Key Contribution
Showed spiking networks with delays have memory capacity far exceeding Hopfield networks. Influenced understanding of hippocampal replay and cortical sequence memory. Used in modern delay-based SNN architectures.
III
Neuromorphic Hardware Emerges
2006 – 2014
2006 – 2011
IBM TrueNorth
Merolla et al., Science 2014
4096 neurosynaptic cores, 1M programmable spiking neurons, 256M synapses. Event-driven (no clock-based computation between spikes). 70mW inference on a chip the size of a postage stamp.
HardwareIBMDigitalCMOS
Architecture
Massively parallel, event-driven digital design. Each core has 256 neurons × 256 synapses. No shared memory bus — spikes route via an on-chip mesh network. 4096 cores tile seamlessly for larger systems.
Key Achievement
Demonstrated 400× better energy-per-synaptic-op than GPU inference. Used for DARPA SyNAPSE program. Applied to multi-object detection and keyword spotting.
Use Cases
  • Always-on sensor fusion
  • Keyword spotting
  • Low-power edge vision
2009 – 2014
SpiNNaker
Furber et al., Manchester
Many-core ARM-based massively parallel system designed for real-time spiking neural simulation. 1M ARM cores in SpiNNaker-1 machine. Flexible — neurons run as software on cores. HBP flagship platform.
HardwareManchesterFlexibleHBP
Architecture
18 ARM9 cores per chip, custom packet-switched NoC. Neurons modeled in software — can run any neuron model (LIF, H-H, Izhikevich). SpiNNaker 2 (2022): 10 nm, 152 cores per chip, integrated ML accelerator.
Use Cases
  • Brain-scale simulation
  • Robotics control
  • Closed-loop neural interface
  • HBP cortical modeling
2010 – 2015
BrainScaleS
Heidelberg / HBP
Analog neuromorphic hardware — physical circuits implement neuron dynamics in silicon. Runs 10,000× faster than biological real-time. Wafer-scale integration with 200,000 neurons per wafer.
HardwareAnalogAcceleratedHBP
Architecture
AdEx neurons in analog VLSI. Synaptic weights stored on-chip as conductances. BSS-2 (2020s): adds plasticity support (STDP), on-chip learning. Speed advantage enables evolutionary / RL training.
Use Cases
  • Accelerated neural evolution
  • Online STDP learning experiments
  • Closed-loop brain simulation
IV
ANN-to-SNN Conversion & Deep SNNs
2014 – 2019
2014 – 2017
ANN-to-SNN Conversion
Rueckauer / Diehl method
Train a standard deep ANN (ResNet, VGG), then convert ReLU activations to spiking LIF neurons by equating firing rate with activation magnitude. Achieves near-ANN accuracy at the cost of many timesteps (100–1000).
ConversionDeep SNNRate Coding
Key Contribution
First scalable path to deep SNNs with competitive accuracy. Weight normalization and threshold balancing are critical. Achieves <1% gap from ANN on CIFAR-10. Main drawback: requires many timesteps → high latency, partially negating energy savings.
Use Cases
  • Image classification on neuromorphic chips
  • Deploying ANNs on Loihi / TrueNorth
  • Energy-efficient inference
2018
Intel Loihi 1
Davies et al., IEEE Micro 2018
Intel’s first neuromorphic research chip. 128 cores, 131,072 neurons, 130M synapses. On-chip STDP learning engine — programmable learning rules in hardware. 1,000× better energy / inference vs CPU on sparse workloads.
HardwareIntelOn-chip Learning
Architecture
Hierarchical mesh NoC. Learning engine supports 3-factor (reward-modulated) STDP. Compartment model supports multi-compartment neurons. SDK: NxSDK (Python). Used for SLAM, sparse coding, constraint satisfaction.
Key Results
SLAM on Loihi used 100× less energy than equivalent GPU. Olfactory system analogy achieved one-shot odor learning. Keyword spotting with 1.4µJ per inference.
Use Cases
  • Sparse coding
  • SLAM / robotics
  • One-shot learning
  • Constraint satisfaction
2018 – 2019
Surrogate Gradient Methods
SuperSpike / SLAYER / STBP
Replace the non-differentiable spike function with a smooth surrogate during backpropagation (sigmoid, fast sigmoid, arctan). Enables direct training of deep SNNs with standard backprop — the turning point for SNN performance.
TrainingSurrogate GradientBPTTDeep SNN
Key Contribution
SuperSpike (Zenke & Ganguli 2018): theoretical foundation. SLAYER (Shrestha & Orchard 2018): efficient BPTT for temporal SNNs on GPU. STBP (Wu et al. 2018): spatial-temporal backprop enabling VGG / ResNet-depth SNNs. Closed the accuracy gap with ANNs to <0.5% on CIFAR-10 within 2 years.
Use Cases
  • Event camera processing
  • N-MNIST / N-CALTECH classification
  • Audio keyword spotting
  • Gesture recognition
V
Event Cameras, Online Learning & Bioplausible Backprop
2018 – 2022
2018 – 2020
Event-Based Vision
DVS / DAVIS cameras + SNN
Dynamic Vision Sensors (DVS) fire asynchronous spike events per pixel when brightness changes. Natural SNN input — microsecond temporal resolution, 120dB dynamic range, near-zero power at rest.
SensingEvent CameraAsyncLow Power
Key Contribution
Prophesee, iniVation, Samsung DVS cameras commercially available. SNN + event camera stack achieves object recognition at 10,000 FPS equivalent with µW sensor power. N-MNIST, N-CALTECH101, DVS-Gesture are benchmark datasets.
Use Cases
  • High-speed object tracking
  • Autonomous vehicle corners
  • Gesture recognition
  • Industrial inspection
  • Space debris tracking
2020
e-prop / DECOLLE
Eligibility-trace based online learning
Biologically plausible online learning rules that approximate BPTT using eligibility traces — local synaptic variables tracking recent activity. Enables online credit assignment without storing full activity history.
Learning RuleOnlineBioplausibleRecurrent SNN
Key Contribution
e-prop (Bellec et al. 2020): learning in recurrent SNNs (LSNN) achieves near-BPTT accuracy on speech tasks. DECOLLE: layer-local loss + eligibility traces, fully online. Both enable on-chip learning without backprop infrastructure.
Use Cases
  • On-chip continual learning
  • Speech processing
  • Recurrent temporal tasks
  • Neural prosthetics
2019 – 2021
Spiking ResNet / VGG
SEW-ResNet / MS-ResNet
Deep SNN architectures directly inspired by residual connections. Spike-Element-Wise (SEW) ResNet adds residual pathways that carry binary spike tensors — enabling very deep SNNs without gradient vanishing.
Deep SNNResNetSurrogate Gradient
Key Contribution
SEW-ResNet-18 achieves 63.18% top-1 on ImageNet with T=4 timesteps — closing the ANN accuracy gap significantly. Enabled SNN training on large-scale benchmarks. Opened path to SNN ViTs.
Use Cases
  • ImageNet classification
  • Object detection
  • DVS video understanding
2021
Intel Loihi 2
Intel 4 process (7nm EUV)
10× more neurons (1M), programmable neuron models (any IF variant), graded spikes, new learning engine supporting third-factor neuromodulation. SDK: Lava (open-source). 10× better energy efficiency than Loihi 1.
HardwareIntelProgrammable
Architecture
128 neuromorphic cores + 6 embedded Lakemont x86 cores for host management. Supports dense and sparse connectivity. Graded spikes allow non-binary values. Up to 100 compartments per neuron for complex dendritic computation.
Use Cases
  • Robotic control
  • Scientific computing
  • Sparse optimization
  • Edge AI inference
  • On-chip RL
VI
Spiking Transformers, LLM Convergence & Frontier Hardware
2022 – 2024
2022 – 2023
Spiking Transformers
Spikformer / Spike-driven Transformer
Replace softmax attention with spike-based attention (binary Q, K, V matrices). Spikformer achieves 74.81% top-1 on ImageNet with pure spike-driven computation — no floating-point multiply-accumulate, only additions.
TransformerSpike-drivenMAC-free
Key Contribution
Spiking Self-Attention (SSA) uses binary spike tensors — O(T·N²) ops become simple accumulations. Spike-driven Transformer v2 (2023): 77.1% ImageNet top-1. Meta-SpikeFormer: 80.0% with hierarchical design. MAC-free inference is the key hardware advantage.
Use Cases
  • Energy-efficient vision
  • Neuromorphic chip deployment
  • Mobile / edge vision transformers
2023
SpikingBERT / Spiking LLMs
SpikeGPT / SpikingBERT
First spiking language models. SpikeGPT: autoregressive spiking transformer (GPT-style) trained from scratch. SpikingBERT: BERT converted to spiking — competitive on GLUE benchmarks with binary activations.
NLPLLMSpikingEmergent
Key Contribution
Demonstrates SNNs can operate on sequential language tasks. 8× energy reduction vs BERT on targeted benchmarks. Opens research into neuromorphic LLM inference. Still far behind full-precision LLMs but growing fast.
Use Cases
  • On-device NLP
  • Low-power text classification
  • Edge language inference
2023 – 2024
Hala Point (Intel)
1.15B neuron neuromorphic system
World’s largest neuromorphic system: 1152 Loihi 2 chips, 1.15 billion neurons, 128 billion synapses. Deployed at Sandia National Labs. Demonstrated 20 TOPS/W on sparse workloads.
HardwareIntelScale2024
Key Achievement
Reaches brain-scale neuron count (human cortex ~16B neurons, but Hala Point has more than a mouse brain). 2,800× better energy-delay product than conventional CPU / GPU for SNN workloads. Used for scientific computing and continuous learning research.
Use Cases
  • Brain-scale SNN simulation
  • Scientific sparse computing
  • Continual learning research
  • Edge AI R&D
2023 – 2024
GNN + SNN Convergence
SpikeGCN / Spiking GNN
Run GNN message passing using spiking neurons — sparse, asynchronous propagation over graph edges. SpikeGCN, EfficientSNN on graphs, and GNN inference on Loihi. Enables ultra-low-power graph inference at the edge.
GNNSNNConvergenceEdge AI
Key Contribution
Message passing is naturally sparse and event-driven — maps perfectly to neuromorphic hardware. SpikeGCN achieves competitive node classification. Graph problems on Loihi 2 show 10–100× energy savings vs GPU for sparse graphs. The most natural intersection of graph ML and neuromorphic computing.
Use Cases
  • On-chip knowledge graph inference
  • Molecular property prediction (edge)
  • Sensor network processing
  • Low-power traffic graph inference
2022 – 2024
Memristive / Analog NM Chips
Akida / Xylo / MorphIC
Commercial neuromorphic edge chips: BrainChip Akida (always-on keyword / vision), SynSense Xylo (audio SNNs on hearing aids), MorphIC (EPFL). Also: Samsung HBM+SNN, SK Hynix PIM-SNN prototypes.
CommercialEdgeAnalogAlways-on
Landscape
BrainChip Akida: AKD1000 deployed in hearing aids and IoT. SynSense Xylo: 1µW always-on audio SNN. MorphIC: TSMC 65nm, 2k neurons, on-chip STDP. IBM analog AI chip: PCM crossbar for in-memory ANN (approaching SNN energy). Mythic AI, Syntiant for non-spiking edge inference comparison.
Use Cases
  • Hearing aids / wearables
  • Always-on keyword spotting
  • IoT anomaly detection
  • Drone navigation

Quick reference

SNN vs ANN vs Neuromorphic hardware

The clearest way to read the field is to compare it against what enterprise teams already know. Classical ANNs dominate production today; SNNs and neuromorphic hardware are advancing fastest where energy, latency, or sparsity break the conventional model.

“The brain runs on 20 watts. Every serious bid for sustainable AI eventually has to ask why our silicon doesn’t.”

AI should be a tool, not a takeover.

Neuromorphic computing is one of the most promising paths to AI that runs on the energy budget of a brain rather than a datacenter. Whether it reshapes the enterprise stack — and how leaders govern it when it does — will define the next decade of alignment work.
