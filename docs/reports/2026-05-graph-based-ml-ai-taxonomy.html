Why a graph view of ML matters

Most enterprise data is already a graph.

Customers, transactions, suppliers, molecules, networks, knowledge bases — the value isn't in any single record, it's in the relationships between them. Understanding the lineage of graph-based ML is how enterprise leaders decide which architectures to invest in, which vendors to trust, and where the next wave of AI capability is heading.

Lens 01

A map of architectural lineage

Every modern graph model traces back to a small set of foundational ideas. Seeing the lineage clarifies which approaches are mature, which are still experimental, and which are converging into tomorrow's foundation models.

Lens 02

The bridge to LLMs

Knowledge graphs, GraphRAG, and graph foundation models are the connective tissue between structured enterprise data and large language models. They are how LLMs become trustworthy, grounded, and auditable.

Lens 03

A vocabulary for governance

Conversations about agent governance, drug discovery, fraud detection, and protein design all rely on this vocabulary. This reference exists so leaders can read the room when their teams say “GNN,” “MPNN,” or “Evoformer.”

The evolutionary lineage

Seven eras of graph-based AI

Each card is a key architecture or paradigm. Click to reveal its innovation, real-world uses, and limitations. The spine marks the chronological backbone — the eras above it group related advances.

Era I · Classical Foundations · 1980s–2000s
~1988
Bayesian Networks
BN · DAG Models
Directed acyclic graphs encoding probabilistic conditional dependencies between variables. Pearl's foundational framework for reasoning under uncertainty.
Probabilistic Directed Inference
Key Innovation
Compact representation of joint probability distributions via conditional independence. Enables exact and approximate Bayesian inference (belief propagation, junction trees).
Use Cases
  • Medical diagnosis
  • Fault detection
  • Spam filtering
  • Risk analysis
  • Gene regulatory networks
Limitation
Exponential complexity in dense graphs; requires expert-defined structure or structure learning from data.
~1994
Markov Random Fields
MRF · Undirected GM
Undirected graphical models capturing symmetric dependencies. Boltzmann machines are a neural instance. Widely used in computer vision for structured prediction.
Probabilistic Undirected Energy-Based
Key Innovation
Gibbs distributions allow modeling spatial/temporal correlations without causal direction. Loopy belief propagation enables approximate inference.
Use Cases
  • Image segmentation
  • Stereo vision
  • Denoising
  • Social network analysis
  • NLP tagging
~1998
Kernel Methods on Graphs
Graph Kernels
Measure graph similarity by counting shared substructures — walks, paths, subtrees, Weisfeiler–Lehman patterns. Plugged into SVMs for graph classification.
Classical ML Graph Classification
Key Innovation
The WL kernel (2011) became the gold standard, later inspiring the theoretical understanding of GNN expressiveness via WL graph isomorphism tests.
Use Cases
  • Molecule classification
  • Protein function prediction
  • Malware detection
  • Social network classification
Era II · Deep Graph Learning Emerges · 2013–2017
2013 – 2014
DeepWalk / Node2Vec
Random-Walk Embeddings
Apply Word2Vec–style skip-gram to random walk sequences on graphs. Node2Vec adds biased walks (BFS/DFS) for flexible neighborhood exploration.
Scalable Transductive Embeddings
Key Innovation
First scalable approach to learn continuous node representations. Inspired LINE (2015) and a wave of network embedding methods (SDNE, HOPE).
Use Cases
  • Link prediction
  • Community detection
  • Recommender systems
  • Knowledge graph completion
Limitation
Transductive (can't generalize to unseen nodes without retraining). No feature aggregation.
2014
Spectral Graph CNN
Spectral GCN (Bruna et al.)
Defines convolution in the graph Fourier domain via eigendecomposition of the graph Laplacian. Theoretically sound but expensive at scale.
Spectral Convolution Foundational
Key Innovation
Bridged signal processing on graphs with deep learning. Defined the theoretical foundation for all spectral GNNs. ChebNet (2016) made it practical with polynomial approximations.
Use Cases
  • Node classification
  • Graph classification
2017
Graph Convolutional Network
GCN (Kipf & Welling)
Simplifies spectral convolution to a first-order Chebyshev approximation. Each node aggregates a normalized mean of its neighbors' features. Seminal paper with 20,000+ citations.
Spectral Semi-Supervised Node Classification
Key Innovation
Layer-wise propagation rule with the renormalization trick. Practical, fast, effective on citation networks. Became the baseline for the entire field.
Use Cases
  • Citation network classification
  • Knowledge graph reasoning
  • Drug discovery
  • Traffic prediction
Limitation
Fixed graph at training (transductive). Deep stacking dilutes features (over-smoothing). Uniform neighbor weighting.
Era III · Spatial & Message-Passing Paradigm · 2017–2019
2017
Message Passing Neural Net
MPNN (Gilmer et al.)
Unifying framework: nodes pass messages to neighbors, aggregate, and update their own state. Most spatial GNNs are special cases of MPNN. Origin paper from Google Brain on molecular property prediction.
Spatial Unifying Framework Chemistry
Key Innovation
Message function M, aggregation Σ, and update function U define a complete GNN paradigm. Enabled systematic comparison and design of new architectures.
Use Cases
  • Quantum chemistry (QM9 benchmark)
  • Molecular property prediction
  • Drug–target interaction
2017
Graph Attention Network
GAT (Veličković et al.)
Introduces learnable attention coefficients for neighbor aggregation. Nodes attend differentially to neighbors — resolving GCN's uniform-weighting problem.
Attention Inductive Multi-head
Key Innovation
Multi-head attention on graph neighborhoods. No eigendecomposition required. GAT v2 (2022) fixes expressiveness issues where attention collapsed in the original formulation.
Use Cases
  • Social network analysis
  • Protein interface prediction
  • NLP dependency parsing
  • Recommendation
2017
GraphSAGE
SAmple & aggreGatE
Inductive learning by sampling fixed-size neighborhoods and aggregating (mean / max / LSTM). First GNN to scale to massive graphs via mini-batch training.
Scalable Inductive Sampling
Key Innovation
Mini-batch training on subgraphs via neighborhood sampling. Enables deployment on billion-node graphs (PinSage used it at Pinterest scale).
Use Cases
  • Pinterest recommendations (PinSage)
  • E-commerce graphs
  • Fraud detection at scale
  • Citation networks
2019
Graph Isomorphism Network
GIN (Xu et al.)
Theoretically proves GNNs are at most as powerful as the Weisfeiler–Lehman graph isomorphism test. GIN achieves this upper bound using injective aggregation (sum + MLP) — the most expressive standard MPNN.
Spatial Theory Expressiveness
Key Innovation
Formal characterization of GNN power. Showed mean/max aggregation is strictly weaker than sum. Opened research into higher-order GNNs (k-WL, NGNN) that go beyond WL.
Use Cases
  • Graph classification benchmarks
  • Molecular property prediction
  • Combinatorial optimization
Era IV · Generative & Knowledge Graph Models · 2018–2020
2018
Graph Variational Autoencoder
GVAE / VGAE
Combines a GCN encoder with an inner-product decoder in a VAE framework. Learns latent representations of graph structure for generation and link prediction.
Generative Latent Space Unsupervised
Key Innovation
First principled probabilistic generative model for graphs. Extended to conditional CVAE for molecule generation (JT-VAE with junction-tree decoder).
Use Cases
  • De novo drug design
  • Link prediction
  • Network completion
  • Molecule generation
2018 – 2019
Knowledge Graph Embeddings
TransE / RotatE / ComplEx
Learn entity and relation embeddings in continuous space. TransE models relations as translations; RotatE uses rotation in complex space; ComplEx handles symmetric/antisymmetric relations.
Knowledge Graphs Link Prediction Reasoning
Key Innovation
Score function h+r≈t (TransE). Later models — RGCN, CompGCN, NBFNet — add neural power. Foundation for KGQA and multi-hop reasoning.
Use Cases
  • Freebase / Wikidata completion
  • Question answering over KGs
  • Biomedical KG (DRKG)
  • Recommender systems
  • Entity alignment
2018
MolGAN / CGVAE
Graph GAN · CGVAE
MolGAN uses GAN + RL reward for direct molecule generation. CGVAE conditions on valid chemistry constraints. Both produce molecular graphs directly as adjacency/feature matrices.
Generative Drug Discovery GAN
Key Innovation
RL reward shaping for chemical validity and QED drug-likeness. Opened the field of deep generative chemistry. Later superseded by flow-based and diffusion models.
Use Cases
  • De novo drug design
  • Material discovery
  • Molecular optimization
Era V · Scaling, Transformers & Heterogeneous Graphs · 2019–2022
2019
Heterogeneous GNN
HAN / HGT / RSAGE
Extend GNNs to graphs with multiple node and edge types. HAN uses type-level attention; HGT uses Transformer-style attention with per-type projection matrices.
Heterogeneous Multi-relational Attention
Key Innovation
Meta-path attention (HAN) and type-specific transformations (HGT). Essential for real-world graphs with semantic edge types (user–item–category, gene–disease–drug).
Use Cases
  • E-commerce recommendation
  • Academic graph analysis
  • Biomedical networks
  • Financial fraud detection
2019 – 2020
Scalable GNN Training
ClusterGCN / GraphSAINT / SIGN
ClusterGCN partitions graphs into clusters for mini-batches. GraphSAINT samples subgraphs with importance weighting. SIGN precomputes graph operators without sampling.
Scalable Industrial Mini-batch
Key Innovation
Enabled GNN training on ogbn-papers100M (100M nodes, 1B edges). These techniques power production GNNs at Google, Meta, Alibaba, and X.
Use Cases
  • Web-scale recommendation
  • Large-scale fraud detection
  • Billion-node social graphs
2020 – 2021
Graph Transformer
GT / Graphormer / SAN
Apply full Transformer self-attention to graph nodes, encoding structure via positional encodings (Laplacian eigenvectors, random walk scores, shortest-path distances).
Attention Transformer Long-range
Key Innovation
Graphormer won OGB-LSC (2021) molecular challenge. Overcomes over-squashing by allowing any-to-any attention. GPS (General, Powerful, Scalable) unifies MPNN + attention.
Use Cases
  • Molecular property prediction
  • Quantum chemistry
  • Traffic forecasting
  • Code analysis
2021 – 2022
Graph Diffusion Models
GDSS / DiGress / DiffSBDD
Apply score-based diffusion (DDPM/DDIM) to graphs. Jointly diffuse node features and adjacency. DiffSBDD targets structure-based drug design in 3D protein pockets.
Generative Diffusion 3D Molecular
Key Innovation
State-of-the-art molecular generation quality. GEOM-Drug benchmark dominance. DiGress uses discrete diffusion on graph tokens. EquiDiff and Torsional Diffusion add 3D equivariance.
Use Cases
  • Structure-based drug design
  • Protein–ligand docking
  • Material generation
  • Molecule optimization
Era VI · Physics, Biology & Geometric GNNs · 2020–2023
2021
Equivariant GNNs
EGNN / SE(3)-Trans / NequIP
Enforce equivariance to 3D rotations and translations (SE(3) / E(3) symmetry). Node features transform as geometric tensors — essential for molecular and physical simulations.
Geometric Physics Equivariant
Key Innovation
NequIP outperforms force fields with 1000× less data. MACE (2022) uses higher-body-order equivariant features. Foundational for universal ML potentials (MACE-MP-0).
Use Cases
  • Molecular dynamics simulation
  • Protein structure prediction
  • Crystal property prediction
  • Quantum chemistry
2021 – 2022
AlphaFold 2 — Evoformer
Evoformer (DeepMind)
Evoformer treats the amino acid distance map as a graph. Axial attention on pairwise edges + MSA rows. The most impactful applied GNN to date — predicted ~200M protein structures.
Structural Biology Attention Landmark
Key Innovation
Triangular multiplicative updates on pair representations enforce geometric consistency. Iterative recycling. AlphaFold 3 extends to nucleic acids and small molecules using a diffusion head.
Use Cases
  • Protein structure prediction
  • Drug target discovery
  • Enzyme engineering
  • Antibody design
2022
Temporal & Dynamic GNNs
TGN / JODIE / DyRep
Model evolving graphs with time-stamped edges. TGN maintains per-node memory updated by temporal attention over interaction history. Essential for streaming graph data.
Temporal Streaming Dynamic
Key Innovation
Continuous-time dynamic graph processing. Memory module + graph attention enables future link prediction and event-time prediction. Used in real-time fraud detection on transaction graphs.
Use Cases
  • Real-time fraud detection
  • User behavior modeling
  • Temporal knowledge graphs
  • Financial transaction networks
Era VII · Graph Foundation Models & LLM Integration · 2023–Present
2023 – 2024
Graph Foundation Models
OFA / PRODIGY / GraphFM
Pre-train a single GNN on diverse graph datasets for zero / few-shot transfer. OFA (One For All) uses text-attributed graphs; PRODIGY uses in-context learning on graphs.
Foundation Model Transfer Learning Zero-shot
Key Innovation
Cross-domain pre-training on heterogeneous graph types. Prompted with task descriptions (text). Bridges the “no universal graph vocabulary” challenge that prevented GPT-style scaling for graphs.
Use Cases
  • Few-shot node classification
  • Cross-dataset transfer
  • Multi-task graph learning
2023 – 2024
LLM + Graph RAG
GraphRAG / G-Retriever
Augment LLM reasoning with structured knowledge-graph retrieval. GraphRAG (Microsoft) builds community summaries from KGs for global QA. G-Retriever uses a GNN for subgraph retrieval.
LLM + Graph RAG Knowledge
Key Innovation
Knowledge graphs provide structured, verifiable context to LLMs — reducing hallucination. GraphRAG enables multi-hop reasoning impossible with flat vector RAG. Widely used in enterprise AI.
Use Cases
  • Enterprise knowledge QA
  • Multi-hop reasoning
  • KGQA (Freebase, Wikidata)
  • Medical literature analysis
2024
Graph Neural Diffusion / PDE-GNN
GRAND / GraphBelief
Reframe GNN propagation as discretizing continuous PDEs (heat / wave equations) on graphs. Enables principled design of deep, stable GNNs that resist over-smoothing.
Theory PDE Deep GNNs
Key Innovation
GRAND interprets message-passing layers as Euler steps of a diffusion PDE. Continuous-depth GNNs via Neural ODEs on graphs. Alleviates over-smoothing with nonlinear diffusion. Unifies many GNN architectures.
Use Cases
  • Long-range graph tasks
  • Deep GNN training
  • Physics simulation
